element日期选择范围限制

element 日期选择范围限制

功能需求

  • 开始时间不得在接口返回时间之前，以及不得在结束时间之后
  • 结束时间不得在当前时间之后，以及不得在开始时间之前

解决

  1. 可以通过设置 picker-options 属性来设置当前时间日期选择器特有的选项
  2. 通过设置 disabledDate 来禁用范围外的时间选项，就可以限制可选的范围了。而 disabledDate 是一个函数，返回 boolean 类型。
